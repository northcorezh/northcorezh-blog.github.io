## 部署Vue环境
Vue官网  
https://cn.vuejs.org/  

```SHELL

# 国内直接使用npm 的官方镜像是非常慢的，这里推荐使用淘宝 NPM 镜像
npm  install  -g  cnpm  --registry=https://registry.npm.taobao.org

# 第二步项目初始化
# 安装vue-cli
cnpm install vue-cli -g

# 需要检查vue
vue list

# 选定路径，新建vue项目
vue init webpack xxx 项目名称

# 安装下依赖
cd xxx
cnpm install

# 运行Vue
cnpm run dev

访问 http://localhost:8080/

```
